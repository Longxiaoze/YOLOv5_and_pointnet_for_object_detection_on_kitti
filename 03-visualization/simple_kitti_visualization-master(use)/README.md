# simple Kitti visualization scripts

This repository contains simple visualization scripts 
for [Kitti](http://www.cvlibs.net/datasets/kitti/) 
and [Semantic Kitti](http://www.semantic-kitti.org/) dataset.
 
## Requirements:
- python>=3.6
- numpy
- matplotlib
- mayavi
- skimage
- seaborn
